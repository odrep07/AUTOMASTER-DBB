# AutoMaster – Sistema de Gestão de Oficina Mecânica
Repositório contendo scripts SQL para criação, povoamento e manipulação do banco de dados da oficina AutoMaster.

Estrutura do Projeto
Este projeto implementa o banco relacional modelado nas Experiências Práticas anteriores, incluindo:

- Modelo lógico normalizado
- Tabelas com PK, FK e integridade referencial
- Inserção de dados reais
- Consultas SQL (SELECT)
- Manipulação de dados (UPDATE / DELETE)

Tecnologias utilizadas
- MySQL Workbench 8+ **ou** PostgreSQL (PGAdmin)
- SQL (DDL + DML)
- GitHub

 Arquivos incluídos
- `01_create_tables.sql` → Criação das tabelas do sistema  
- `02_insert_data.sql` → Inserção de dados iniciais  
- `03_select_queries.sql` → Consultas SELECT  
- `04_updates_deletes.sql` → Comandos UPDATE e DELETE
